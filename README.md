# C & MySQL: Getting started

Everything you need to know to connect C with your MySQL

## Prerequisite
* OS: Linux
* Additional library: `libmysqlclient`
```
sudo apt install default-libmysqlclient-dev # Run this to download libmysqlclient
```

## How to run
Run `makefile`
```
make
```
Run program
```
.\main
```
